Traditional Resume Checklist

Resume Sections:
  • Name, Phone Number, Email, Location (City,State), Position Title
  • Work Status (U.S. Citizen, Green Card Holder, Authorized to work in U.S. for any employer without sponsorship)
  • Professional Summary (Around 1 page long)
  • List of Technical Skills
  • Certifications (if you have any)
  • Experience
    • Label area for position title, client name, client location, job duration
    • Comprehensive list of responsibilities for each job description
    • 3 job experiences
  • Location history → should be provided in month, year, city, state format (please include this separately on last page of your resume)
  • Resume should be around 4-5 pages long
Final Resume Edit:
  • Make sure there is consistent formatting
    • Check for consistent font, font size, font color, font bolding, font capitalization
    • Check for consistent spacing
  • Make sure resume is written in consistent tense
    • Previous jobs should be written in past tense (you may also write your current position in past tense)
  • Proper Capitalization of Technical Terms
    • Refer to names of software on Google for proper capitalization of technical terms
